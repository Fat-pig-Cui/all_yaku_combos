## 条件役展开

### 1番

|    | 备注 |
|:--:|:--:|
| 枪杠 | 荣  |
| 岭上 | 露  |
| 海底 | 露  |
| 河底 | 荣  |

副露状态下, "枪杠"无法与"三暗"复合

故对于"枪杠", 剔除含有"三暗"的, 赋值给 `table_2`

| 番数 | 总数 | 差值 | 
|:--:|:--:|:--:|
| 1  | 1  |    |
| 2  | 5  |    |
| 3  | 12 | 1  |
| 4  | 19 | 5  |
| 5  | 24 | 12 |
| 6  | 23 | 19 |
| 7  | 16 | 24 |
| 8  | 9  | 23 |
| 9  | 3  | 16 |
| 10 | 0  | 9  |
| 11 | 0  | 3  |

因为所有的能复合的条件役都是1番, 故可以 `new_table[i + 1] += table[i] * 3`, 更新 `new_table`

使用3次 `table` 和1次 `table_2`, 得到结果
