# 所有非役满役种组合之: 无断幺的平和专题

平和因为它对牌型有整体的要求, 这里把它单独拎出来

为了防止重复, 这里只涉及不含断幺的部分, 含有的部分可以看 [一般形断幺九专题](../一般形断幺九专题)

- 打"*"表示可以复合但一定会伴随存在其他役种, 不符合条件
- 打"&"表示只有三麻的拔北可以做到, 不计数
