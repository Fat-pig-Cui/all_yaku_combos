## 条件役专题详解

- "枪杠"无法与"七对", "对对", "混老", "二杯"复合, 若能与"三暗"复合一定是门清状态
- "岭上"与"平和", "七对", "二杯"能复合只有三麻的拔北能做到
- "自摸"与"对对"无法复合
- "两立", "一发"无法同时与"三杠"复合
