# 所有非役满役种组合一览之: 一般形断幺专题

除去 [七对子](../七对子专题.md) 和 [小三元](../小三元专题) 剩下的部分仍然实在是太多, 又分成了一般形断幺专题

因为是一般形, 所以不再重复考虑七对子

打"^"表示只有副露使得食下役减番才满足条件
