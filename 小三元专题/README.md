# 所有非役满役种组合一览之: 小三元专题

小三元因为它出现一定还会伴随两个三元牌, 这里把它单独拎出来, 并在后面的一览中省略两个三元牌

- 打"*"表示可以复合但一定会伴随存在其他役种, 不符合条件
- 打"^"表示只有副露使得食下役减番才满足条件

由于小三元部分就已经有三种组合, 故数量要乘以3
