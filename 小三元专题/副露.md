## 条件役展开

### 1番

|    |
|:--:|
| 枪杠 |
| 岭上 |
| 海底 |

"枪杠"不能与"对对"复合, 在副露的情况下也不能与"三暗"复合,

故对于"枪杠", 剔除含有"对对"和"三暗"的, 赋值给 `table_2`

| 12 | 0  | 9  |
| 13 | 0  | 7  |

| 16 | 0  | 1  |

因为所有的能复合的条件役都是1番, 故可以 `new_table[i + 1] += table[i] * 3`, 更新 `new_table`
