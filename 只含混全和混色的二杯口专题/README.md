# 所有非役满役种组合一览之: 只含混全和混色的二杯口专题

因为能与二杯口复合的役很少, 而且大部分已经在之前的专题中用到了, 这里手牌役就只剩 混全带幺九 和 混一色 了

其他能复合的役种: [断幺](../一般形断幺专题), [平和](../无断幺的平和专题), 
[清一色](../无断幺平和的一般形清一色专题), [纯全带幺九](../无平和清一色的纯全带幺九专题)

打"&"表示只有三麻的拔北可以做到, 不计数
